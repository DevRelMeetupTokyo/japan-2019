# DevRel/Japan Conference 公式ウェブサイト

## Getting started

```sh
$ cd /path/to/dir
$ git clone git@github.com:DevRelMeetupTokyo/japan-2019.git
$ cd japan-2019
$ docker-compose up -d
```

Now browse to http://localhost:4000/japan-2019 .

## デプロイについて

masterブランチにマージされたら、自動で[本番サイト](https://devrel.tokyo/japan-2019/)に反映されます。
